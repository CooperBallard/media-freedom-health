# Data Sources

Raw source files are too large for GitHub. Download them from the links below. A pre-merged dataset (`merged_panel.csv`) is included in this folder for convenience.

## Merged Dataset (included)

- **File:** `merged_panel.csv`
- **Description:** V-Dem governance indicators merged with World Bank health expenditure data, 2000–2024
- **Unit of analysis:** Country-year (4,457 rows)

## V-Dem Country-Year Full+Others (v16)

- **Source:** [V-Dem Institute](https://v-dem.net/data/the-v-dem-dataset/)
- **File:** `V-Dem-CY-Full+Others-v16.csv`
- **Coverage:** 170+ countries, 1900–2025
- **Key variables:** `v2mecenefm` (media censorship), `v2meslfcen` (self-censorship), `v2meharjrn` (journalist harassment), `v2csreprss` (civil society repression), `v2x_freexp_altinf` (freedom of expression index), `e_gdppc`, `e_pelifeex`, `e_wbgi_rle`

## World Bank World Development Indicators

- **Source:** [World Bank](https://data.worldbank.org/indicator/SH.XPD.CHEX.PC.CD)
- **Indicator:** Current health expenditure per capita, USD (`SH.XPD.CHEX.PC.CD`)
- **Coverage:** All countries, 2000–2023
